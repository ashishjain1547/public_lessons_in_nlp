In this post, we share a minimal YAML file to setup an environment for running SOAN project.


name: soan
channels:
  - conda-forge
dependencies:
  - pandas
  - seaborn
  - scikit-learn
  - matplotlib
  - emoji
  - pattern
  - wordcloud
  - palettable
  - ipykernel
  - jupyter
